##Narrative-Emotion-Classifier

This project explores Narrative-Aware Emotion Classification by combining transformer-based models with emotion transition features.

Unlike traditional approaches that classify emotions at the sentence level, this system models how emotions evolve across sequences of text. It leverages a fine-tuned RoBERTa model on the GoEmotions dataset for multi-label emotion prediction and enhances it with narrative-level features such as emotion polarity shift, dominant emotion progression, and emotional volatility.

The goal is to improve contextual understanding of emotional flow in conversations and narratives.
